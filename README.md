# Drag n Drop - Template V2
### Build With : React.js - Next.js - TypeScript - TailwindCSS - React-Beautifull-DnD
        
 User can :  
 - Drag and Drop List 
 - Drag and Drop Card 
 - Add List 
 - Add Card 
 - Rename List
 - Rename Card 
 - Remove List 
 - Remove Card 
 - Move List 
 - Move Card
